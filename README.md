# token_handler
ஒற்றை வரி சரத்தை தனித்தனி தொகுதிகளாய் பிரித்து கையாளக்கூடிய ஒரு சி நிரலகம். அந்த கையாளுமையின் அளவு இயங்குநிலையின் போது ஒதுக்கப்பட்டு பின் நிலையான தன்மையோடு இருக்கும்.

A C library to split a single line string into many individual tokens. The handler's size is dynamically allocated and is fixed once allocated.
